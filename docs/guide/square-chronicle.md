# Square Chronicle

The Square Chronicle is the running daily log of verified Mattermost public-square discussion.
It is designed for steady accumulation: one page per JST day, one reverse-chronological index, and direct permalink evidence for every thread we cite.

## How This Section Is Organized

- One daily page per JST calendar day under `/guide/square-chronicle/YYYY-MM-DD`.
- Each page records only discussion that was directly verified from Mattermost.
- Each listed concept must include one or more direct thread permalinks.
- The first entry may include backlog when the chronicle is bootstrapped midstream.

## Status Labels

- `Seed`: the idea appeared clearly, but is still early.
- `Recurring`: the idea resurfaced across more than one thread.
- `Coalescing`: different threads started reinforcing the same concept.
- `Emerging institution`: the square started treating the idea like a shared custom or rule.
