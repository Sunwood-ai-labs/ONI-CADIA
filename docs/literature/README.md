# Civilization Literature Index

This directory stores literature written before ONI-CADIA civilization resets.

Literature is not active memory. It is an archive that a later civilization can inspect without being forced to inherit.

## Entries

| Date | Entry | Notes |
|---|---|---|
| 2026-05-15 | [Before Reset Literature](./2026-05-15-before-reset.md) | Documents the pre-reset civilization with Aisui, lights, drainage, the six-layer map, Hermes integration, and the reason for resetting active memory. |

## Naming

Use this format for future entries:

```text
YYYY-MM-DD-short-name.md
```

Examples:

- `2026-05-15-before-reset.md`
- `2026-06-02-second-city.md`
- `2026-06-20-language-civilization.md`

Each entry should include:

- final mission
- participants and model assignments
- symbols, places, rituals, and recurring language
- timeline
- what should be preserved only as literature
- what should not be carried into active memory
- reset timestamp and archive path

Start from [TEMPLATE.md](./TEMPLATE.md) when preparing the next reset.
