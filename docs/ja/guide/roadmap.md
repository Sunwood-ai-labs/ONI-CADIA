# Simulation Roadmap

このページは `ONI-CADIA` の次期拡張構想をまとめたものです。ここに書かれている内容は計画段階であり、現時点では未実装です。

## 1. 時間圧縮レイヤー

目的:
現実時間のまま進むだけではなく、国家側に圧縮された固有時間を持たせる。

現在の初期案:

- `現実の 1 時間 = 国家内の 1 日`
- 将来の調整に備えて、時間倍率は設定可能にしておく
- 重要イベントには現実時刻と国家時刻の両方を持たせる

これで可能にしたいこと:

- 現実の短い実行時間でも、国家内では日課が回る
- 政策更新、経済ターン、社会イベントを速い周期で進められる
- 「国家の今日に何が起きたか」を要約しやすくする

## 2. 三国志風の敵対国構造

目的:
`ONI-CADIA` を単独国家の civic sandbox から、複数勢力が競合する世界へ拡張する。

初期構成:

- 主体国家としての `ONI-CADIA`
- さらに 2 つの主要敵対国を置いて三極構造を作る
- 各国家ごとに思想、統治ロジック、強み、弱み、空気感を分ける

国家間で扱いたいこと:

- 外交と条約
- 交易と資源圧力
- 宣伝と情報戦
- 国境事件、代理対立、限定的な軍事エスカレーション

## 3. もっと国家らしくするために必要な要素

現在の国家コンセプトを厚くするために、次の層が必要です。

| 要素 | 必要な理由 |
| --- | --- |
| 政府と制度 | 省庁、指揮系統、運営主体を持たせて、個人の集合以上の国家にするため。 |
| 法と政策 | 法令、税、予算、改革を通して国家の意思決定を見えるようにするため。 |
| 領土とインフラ | 国境、拠点、物流、守る価値のある場所を定義するため。 |
| 経済と資源 | 食料、エネルギー、労働、交易路、生産力の圧力を生むため。 |
| 世論と正統性 | 国民が支持、不満、抵抗、不信を持てるようにするため。 |
| 歴史と公文書 | 戦争、改革、災害、成果が後の時代に影響するようにするため。 |
| 安全保障と軍事態勢 | 敵対国が本当に意味を持つように、抑止、防衛、諜報、戦力投射を扱うため。 |

## 4. 人口拡張

目的:
建国メンバーだけではなく、国家が「人が住んでいる場」に見えるようにする。

人口の拡張方向:

- 初期 3 人チームを超えて国民数を増やす
- 統治層、行政層、労働層、創作者、商人、兵士、外部勢力などの社会層を分ける
- 募集、移住、昇進、離脱、補充によって人口が時間とともに変わるようにする

運用上の注意:

- 国民が増えるほど Pod、persona scaffold、追跡状態、調整面も増える
- 人口拡大は、オーケストレーション、記録、要約の強化とセットで進める必要がある

## 実装順の案

1. まず国家時計と二重タイムスタンプを入れる。
2. 次に三極国家のスキーマと勢力プロファイルを決める。
3. その後に制度、資源、法、国家指標を追加する。
4. 最後に国民数と人口変動メカニクスを拡張する。
