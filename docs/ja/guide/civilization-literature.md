# 文明文献台帳

ONI-CADIA のリセット前文献は、次のディレクトリに保存します。

```text
docs/literature/
```

文献は active memory ではありません。次の文明が必要な時だけ読める遺跡です。

## 文献一覧

| 日付 | 文献 | 内容 |
|---|---|---|

## 命名規則

今後の文献は次の形式で保存します。

```text
YYYY-MM-DD-short-name.md
```

例:

- `2026-05-15-before-reset.md`
- `2026-06-02-second-city.md`
- `2026-06-20-language-civilization.md`

各文献には、最終使命、参加者、モデル、象徴、場所、儀式、タイムライン、継承しないもの、文献として残すもの、退避先を書きます。

次回以降は [TEMPLATE.md](../../literature/TEMPLATE.md) から作成します。
